Class: Parser

Parser

Group: g_constructor

new Parser(options)

Belongs to g_constructor group

**Readability article parsing and bokmarking web API wrapper.** This library eases the pain of performing common tasks while working with the API of readability.com. For more information on reability API visit their website http://www.readability.com/developers/api. As per readability.com website, *"the Parser API is freely available on a limited basis. If you'd like to use the Parser API for commercial use, get in touch with us at licensing@readability.com to learn about our licensing options."*
Parameters:
Name Type Argument Description
options object <optional>
 Configuration for a new instance of the readability. The possible options that you can pass are part of Parser.defaultOptions.
Source:
  • readability.js
Example
// Use Readability to procure the excerpt of an article.
var parser = new require("readability").Parser({
    parserAPIKey: "myapikeyfromreadability.com"
});
parser.parse("http://somedomain.com/somearticle/"'", function (data) {
    console.log(data.excerpt);
});

Members

<static> defaultOptions

Default set of options that are used while communicating with Readability API servers. These are the set of options that can be passed on to a new instance of `Readability` in order to override the defaults.
Properties:
Name Type Description
parserAPIKey string Provide the API key required by readability.com servers for using the API. Though this is optional during instantiating a Parser class, it is a required parameter to be set before actually making any request to the readability servers. If not provided initially, it can be later set using Parser#key method. You can procure your API key from http://www.readability.com/account/api

Methods

Group: g_methods

key(key) → {string}

Belongs to g_methods group

Get or set the parser API key. You can procure your API key from http://www.readability.com/account/api
Parameters:
Name Type Argument Description
key string <optional>
 Parser API key for subsequent use. Do not provide any key if you want to just retrive the current key.
Source:
  • readability.js
Fires:
  • event:#keychange
Returns:
- Readability parser API key being used.
Type
string
Example
// Use Readability to procure the excerpt of an article.
var parser = new require("readability").Parser();

// Lookout for changes to parser API key and log the change.
parser.on("keychange", function (newKey) {
    var currentKey = this.key() || "(blank)"; // retrieve presently set key.
    console.log("Current Key: " + currentKey + ". New Key: " + (newKey || "(blank)"));
});
// Set a new parser key.
parser.key("myapikeyfromreadability.com");
